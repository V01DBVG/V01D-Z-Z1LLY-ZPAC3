# Cookie Kingdom Dash

A lightweight browser game inspired by the lane-dodging and collection style of Cookie Run Kingdom.

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Controls

- Move lanes: `←` / `→` or `A` / `D`
- Jump: `Space`, `W`, or `↑`
- Pause: `P`

## Objective

- Avoid obstacles (spikes and towers)
- Collect mint jellies for bonus points
- Survive as long as possible and beat your best score
